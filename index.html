Info API Google Apps Script

Data login disimpan dan diambil dari Google Sheets melalui Google Apps Script yang berfungsi sebagai API backend.

Ganti URL API di JavaScript dengan URL Web App Anda setelah mendeploy Google Apps Script.

Petunjuk Penggunaan

Langkah 1: Siapkan Google Sheets

Buat spreadsheet baru dengan sheet bernama "Users" dan kolom: Username, Password, Email, Nama Lengkap.

Langkah 2: Buat Google Apps Script

Buka script editor dari spreadsheet, salin kode Apps Script, dan ganti SHEET_ID dengan ID spreadsheet Anda.

Langkah 3: Deploy sebagai Web App

Deploy script sebagai Web App dengan akses "Anyone" (dapat dibatasi sesuai kebutuhan).

Langkah 4: Update URL API

Ganti API_URL di kode JavaScript dengan URL Web App yang Anda dapatkan setelah mendeploy.
